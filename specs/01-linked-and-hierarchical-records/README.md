# Linked and Hierarchical Records

How records in a CESNET Invenio repository (CCMM metadata schema) can relate to one another:

- **Linked records** — *already supported.* Records are related but independent: each is created, published, versioned and access-controlled on its own.
- **Hierarchical (compound) records** — *not yet implemented.* A single logical record split into many for performance/manageability; the **root** record controls publication, versioning and access for the whole tree.
